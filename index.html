HTTP Status Pages

  • 100 Continue 〔進〕 クライアントは、このままリクエスト送ってよかよ。
  • 101 Switching Protocols 〔換〕 サーバーは、クライアントが頼んだ通信プロトコルに切り替えるけんね。
  • 102 Processing 〔処〕 サーバーはリクエストば受け取って、まだ処理しよると。
  • 103 Early Hints 〔先〕 最終レスポンスの前に、関連リソースば先に読み込むための情報ば返すとよ。
  • 104 Upload Resumption Supported 〔再↑〕 サーバーは、途中で止まったアップロードの再開に対応しとるばい。
  • 200 OK 〔良〕 リクエストはちゃんと処理できたばい。
  • 201 Created 〔新〕 リクエストは成功して、新しかリソースが作られたと。
  • 202 Accepted 〔受〕 リクエストは受け付けたばってん、処理はまだ終わっとらんとよ。
  • 203 Non-Authoritative Information 〔間〕 返された情報は、元のサーバーから直接もらったもんやなかと。
  • 204 No Content 〔空〕 リクエストは成功したばってん、返す本文はなかよ。
  • 205 Reset Content 〔戻〕 リクエストは成功したけん、クライアント側の入力画面とかばリセットすると。
  • 206 Partial Content 〔部〕 リソースの一部だけ返しとるとよ。
  • 207 Multi-Status 〔多〕 複数の処理結果ばまとめて返しとるばい。
  • 208 Already Reported 〔済〕 その情報は、もう別の場所で報告済みたい。
  • 226 IM Used 〔差〕 差分処理とかば使った結果ば返しとると。
  • 300 Multiple Choices 〔選〕 リクエストに対して、候補がいくつかあるとよ。
  • 301 Moved Permanently 〔永→〕 リソースはずっと別のURLへ移動しとるばい。
  • 302 Found 〔仮→〕 リソースは一時的に別のURLにあると。
  • 303 See Other 〔見→〕 別のURLばGETメソッドで見に行ってね。
  • 304 Not Modified 〔同〕 リソースは前に取った時から変わっとらんけん、キャッシュ使ってよかよ。
  • 305 Use Proxy 〔代〕 指定されたプロキシば使ってアクセスせんといかんと。
  • 306 Unused 〔休〕 昔は使われよったばってん、今は使われとらんステータスコードたい。
  • 307 Temporary Redirect 〔仮↪〕 一時的に別のURLへリダイレクトするばい。HTTPメソッドは変えんとよ。
  • 308 Permanent Redirect 〔永↪〕 ずっと別のURLへリダイレクトするばい。HTTPメソッドは変えんとよ。
  • 400 Bad Request 〔誤〕 リクエストの書き方や中身に間違いがあって、サーバーは処理できんと。
  • 401 Unauthorized 〔認〕 アクセスするには認証が必要ばい。
  • 402 Payment Required 〔払〕 将来的な支払い処理のために予約されとるステータスコードたい。
  • 403 Forbidden 〔禁〕 権限がなかけん、リクエストは拒否されたと。
  • 404 Not Found 〔迷〕 求められたリソースは見つからんかったばい。
  • 405 Method Not Allowed 〔法×〕 そのリソースでは、そのHTTPメソッドは使えんと。
  • 406 Not Acceptable 〔形×〕 クライアントが受け取れる形式では、レスポンスば返せんと。
  • 407 Proxy Authentication Required 〔代認〕 プロキシサーバーで認証が必要ばい。
  • 408 Request Timeout 〔時〕 クライアントからのリクエスト送信が、時間内に終わらんかったと。
  • 409 Conflict 〔衝〕 リクエストの中身が、今のリソースの状態とぶつかっとるばい。
  • 410 Gone 〔消〕 求められたリソースは、もうずっと使えんごとなっとると。
  • 411 Length Required 〔長?〕 Content-Lengthヘッダーが必要ばい。
  • 412 Precondition Failed 〔前×〕 リクエストで指定された前提条件ば満たしとらんと。
  • 413 Content Too Large 〔大〕 リクエスト本文が大きすぎて、サーバーは処理できんと。
  • 414 URI Too Long 〔URI長〕 リクエストURIが長すぎて、サーバーは処理できんと。
  • 415 Unsupported Media Type 〔媒×〕 リクエスト本文のメディア形式に、サーバーは対応しとらんと。
  • 416 Range Not Satisfiable 〔範×〕 指定された範囲のデータは返せんと。
  • 417 Expectation Failed 〔期×〕 Expectヘッダーで求められた条件ば満たせんかったと。
  • 418 Unused 〔茶休〕 今は使われとらんステータスコードたい。
  • 421 Misdirected Request 〔宛×〕 リクエストが、処理できんサーバーに送られとると。
  • 422 Unprocessable Content 〔意×〕 リクエストの書き方は正しかばってん、中身の意味ば処理できんと。
  • 423 Locked 〔鍵〕 対象リソースはロックされとるばい。
  • 424 Failed Dependency 〔依×〕 依存しとる別の処理が失敗したけん、このリクエストも処理できんと。
  • 425 Too Early 〔早〕 リプレイ攻撃の危険があるけん、サーバーはこのリクエストば処理せんと。
  • 426 Upgrade Required 〔上〕 別のプロトコルへアップグレードせんといかんばい。
  • 428 Precondition Required 〔前!〕 リクエストには前提条件ば入れんといかんと。
  • 429 Too Many Requests 〔多送〕 短時間にリクエストば送りすぎとるばい。
  • 431 Request Header Fields Too Large 〔頭大〕 リクエストヘッダーが大きすぎて、サーバーは処理できんと。
  • 451 Unavailable For Legal Reasons 〔法〕 法的な理由で、リソースば提供できんと。
  • 500 Internal Server Error 〔鯖痛〕 サーバー内部で予想外のエラーが起きたと。
  • 501 Not Implemented 〔未〕 サーバーには、リクエストば処理する機能がまだ実装されとらんと。
  • 502 Bad Gateway 〔門×〕 ゲートウェイやプロキシが、上流サーバーから変な応答ば受け取ったと。
  • 503 Service Unavailable 〔休止〕 サーバーは一時的にリクエストば処理できん状態たい。
  • 504 Gateway Timeout 〔門時〕 ゲートウェイやプロキシが、上流サーバーから時間内に応答ば受け取れんかったと。
  • 505 HTTP Version Not Supported 〔版×〕 サーバーは、リクエストで使われたHTTPバージョンに対応しとらんと。
  • 506 Variant Also Negotiates 〔交×〕 サーバーのコンテンツネゴシエーション設定に問題があると。
  • 507 Insufficient Storage 〔容×〕 サーバーは、リクエスト完了に必要な保存領域ば確保できんと。
  • 508 Loop Detected 〔輪〕 サーバーは、処理中に無限ループば検出したと。
  • 510 Not Extended 〔拡×〕 リクエストば処理するには、必要な拡張情報が足りんと。
  • 511 Network Authentication Required 〔網認〕 ネットワークへアクセスするには認証が必要ばい。
