HTTP Status Pages

  • 100 Continueクライアントは、このままリクエスト送ってよかよ。
  • 101 Switching Protocolsサーバーは、クライアントが頼んだ通信プロトコルに切り替えるけんね。
  • 102 Processingサーバーはリクエストば受け取って、まだ処理しよると。
  • 103 Early Hints最終レスポンスの前に、関連リソースば先に読み込むための情報ば返すとよ。
  • 104 Upload Resumption Supportedサーバーは、途中で止まったアップロードの再開に対応しとるばい。
  • 200 OKリクエストはちゃんと処理できたばい。
  • 201 Createdリクエストは成功して、新しかリソースが作られたと。
  • 202 Acceptedリクエストは受け付けたばってん、処理はまだ終わっとらんとよ。
  • 203 Non-Authoritative Information返された情報は、元のサーバーから直接もらったもんやなかと。
  • 204 No Contentリクエストは成功したばってん、返す本文はなかよ。
  • 205 Reset Contentリクエストは成功したけん、クライアント側の入力画面とかばリセットすると。
  • 206 Partial Contentリソースの一部だけ返しとるとよ。
  • 207 Multi-Status複数の処理結果ばまとめて返しとるばい。
  • 208 Already Reportedその情報は、もう別の場所で報告済みたい。
  • 226 IM Used差分処理とかば使った結果ば返しとると。
  • 300 Multiple Choicesリクエストに対して、候補がいくつかあるとよ。
  • 301 Moved Permanentlyリソースはずっと別のURLへ移動しとるばい。
  • 302 Foundリソースは一時的に別のURLにあると。
  • 303 See Other別のURLばGETメソッドで見に行ってね。
  • 304 Not Modifiedリソースは前に取った時から変わっとらんけん、キャッシュ使ってよかよ。
  • 305 Use Proxy指定されたプロキシば使ってアクセスせんといかんと。
  • 306 Unused昔は使われよったばってん、今は使われとらんステータスコードたい。
  • 307 Temporary Redirect一時的に別のURLへリダイレクトするばい。HTTPメソッドは変えんとよ。
  • 308 Permanent Redirectずっと別のURLへリダイレクトするばい。HTTPメソッドは変えんとよ。
  • 400 Bad Requestリクエストの書き方や中身に間違いがあって、サーバーは処理できんと。
  • 401 Unauthorizedアクセスするには認証が必要ばい。
  • 402 Payment Required将来的な支払い処理のために予約されとるステータスコードたい。
  • 403 Forbidden権限がなかけん、リクエストは拒否されたと。
  • 404 Not Found求められたリソースは見つからんかったばい。
  • 405 Method Not Allowedそのリソースでは、そのHTTPメソッドは使えんと。
  • 406 Not Acceptableクライアントが受け取れる形式では、レスポンスば返せんと。
  • 407 Proxy Authentication Requiredプロキシサーバーで認証が必要ばい。
  • 408 Request Timeoutクライアントからのリクエスト送信が、時間内に終わらんかったと。
  • 409 Conflictリクエストの中身が、今のリソースの状態とぶつかっとるばい。
  • 410 Gone求められたリソースは、もうずっと使えんごとなっとると。
  • 411 Length RequiredContent-Lengthヘッダーが必要ばい。
  • 412 Precondition Failedリクエストで指定された前提条件ば満たしとらんと。
  • 413 Content Too Largeリクエスト本文が大きすぎて、サーバーは処理できんと。
  • 414 URI Too LongリクエストURIが長すぎて、サーバーは処理できんと。
  • 415 Unsupported Media Typeリクエスト本文のメディア形式に、サーバーは対応しとらんと。
  • 416 Range Not Satisfiable指定された範囲のデータは返せんと。
  • 417 Expectation FailedExpectヘッダーで求められた条件ば満たせんかったと。
  • 418 Unused今は使われとらんステータスコードたい。
  • 421 Misdirected Requestリクエストが、処理できんサーバーに送られとると。
  • 422 Unprocessable Contentリクエストの書き方は正しかばってん、中身の意味ば処理できんと。
  • 423 Locked対象リソースはロックされとるばい。
  • 424 Failed Dependency依存しとる別の処理が失敗したけん、このリクエストも処理できんと。
  • 425 Too Earlyリプレイ攻撃の危険があるけん、サーバーはこのリクエストば処理せんと。
  • 426 Upgrade Required別のプロトコルへアップグレードせんといかんばい。
  • 428 Precondition Requiredリクエストには前提条件ば入れんといかんと。
  • 429 Too Many Requests短時間にリクエストば送りすぎとるばい。
  • 431 Request Header Fields Too Largeリクエストヘッダーが大きすぎて、サーバーは処理できんと。
  • 451 Unavailable For Legal Reasons法的な理由で、リソースば提供できんと。
  • 500 Internal Server Errorサーバー内部で予想外のエラーが起きたと。
  • 501 Not Implementedサーバーには、リクエストば処理する機能がまだ実装されとらんと。
  • 502 Bad Gatewayゲートウェイやプロキシが、上流サーバーから変な応答ば受け取ったと。
  • 503 Service Unavailableサーバーは一時的にリクエストば処理できん状態たい。
  • 504 Gateway Timeoutゲートウェイやプロキシが、上流サーバーから時間内に応答ば受け取れんかったと。
  • 505 HTTP Version Not Supportedサーバーは、リクエストで使われたHTTPバージョンに対応しとらんと。
  • 506 Variant Also Negotiatesサーバーのコンテンツネゴシエーション設定に問題があると。
  • 507 Insufficient Storageサーバーは、リクエスト完了に必要な保存領域ば確保できんと。
  • 508 Loop Detectedサーバーは、処理中に無限ループば検出したと。
  • 510 Not Extendedリクエストば処理するには、必要な拡張情報が足りんと。
  • 511 Network Authentication Requiredネットワークへアクセスするには認証が必要ばい。
